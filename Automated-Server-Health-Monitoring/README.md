
# Automated Server Health Monitoring

This project is a Bash-based server monitoring script that logs CPU, memory, and disk usage every 5 minutes. It can also send alerts if thresholds exceed, helping in proactive system management.

## Features
- CPU, Memory, Disk usage logging
- Cron job automation (every 5 minutes)
- Easy to extend with email or Slack alerts
- Perfect for Linux servers

   ```bash
   chmod +x server_monitor.sh
